# Payclip

Payclip ? Because MyCecurity !

## Get started

```
# npm i
# npm link
```

## Usage

```
Usage: payclip [options]

Download latest Payslip

Options:
  -e, --email <email>
  -p, --password <password>
  -l, --last <n>             Download last N payslips (default: 1)
  -h, --help                 display help for command
```
